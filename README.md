# Skill Bridge

Skill Bridge is a role-based skill gap analysis system developed by Team Phoenix.
The project helps students and fresh graduates identify missing skills for
specific industry roles and provides a structured learning roadmap.

## Team
Phoenix

## Features
- Role-based skill gap analysis
- Readiness percentage calculation
- Learning roadmap generation
- Chatbot-style guided flow (SkillBridge)

## Tech Stack
- Python
- Streamlit
- Rule-based skill matching
















## To Start
pip install -r requirements.txt
streamlit run app.py
